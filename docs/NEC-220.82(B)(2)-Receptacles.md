# NEC 220.82(B)(2) – Small-Appliance Branch Circuits

## The Imperial Truth

**220.82(B)(2) is NOT general-use receptacles.**

Those are already imprisoned inside **220.82(B)(1)** — the flat **3 VA per square foot** of habitable floor area. That single number swallows **all** ordinary receptacles in bedrooms, living rooms, hallways, garages (when counted), etc. You do **not** count them again. Ever.

**220.82(B)(2) is the Small-Appliance Branch Circuits** — the **dedicated, high-demand receptacle circuits** required by **210.11(C)(1)**.

---

## Exact Code Language (2023/2026 NEC)

### 220.82(B)(2) Small-Appliance Branch Circuits

> **1500 volt-amperes for each 2-wire small-appliance branch circuit as required by 210.11(C)(1).**

| Parameter | Requirement |
|-----------|-------------|
| **VA per Circuit** | 1500 VA (fixed, no measurement) |
| **Minimum Circuits** | 2 per 210.11(C)(1) |
| **Voltage** | 120V |
| **Amperage** | 20A |
| **Wire** | 12 AWG minimum |
| **Receptacles** | 20A rated (5-20R) |

### What These Circuits Must Serve

Per **210.52(B)**, the small-appliance branch circuits must serve:
- Kitchen countertop receptacles
- Pantry receptacles
- Dining room receptacles
- Breakfast nook receptacles

### What These Circuits MUST NOT Serve

- Lighting outlets (except kitchen/dining)
- Receptacles in other rooms
- Garbage disposal
- Dishwasher
- Range hood (unless it's a receptacle-outlet type)

---

## Why the Empire Separated Them

The **3 VA/ft²** in **(B)(1)** assumes light, occasional use — lamps, phone chargers, TVs, etc.

The **small-appliance circuits** assume **continuous heavy use** in the kitchen — toasters, blenders, coffee makers, microwaves, air fryers, instant pots, and every other kitchen appliance that could overload a normal circuit.

The code forces you to count them at **full 1500 VA strength** so the service can survive a **Saturday morning breakfast invasion** while the coffee brews, toast pops, and kids plug in devices.

## Integration with Demand Factor

After summing **(B)(1) + (B)(2) + (B)(3) + all fixed appliances**:

```
Raw General Load = lighting + small-appliance + laundry + fixed appliances

Demand Factor (220.82(B)):
  First 10,000 VA     → 100%
  Remainder           → 40%

Demanded Load = 10,000 + (Raw General - 10,000) × 0.40
```

## Common Violations (Inspector Wrath)

### ❌ Violation 1: Counting General Receptacles Separately
```
Wrong: 2,800 sq ft × 3 VA + 50 receptacles × 180 VA = DOUBLE COUNTING
Right: 2,800 sq ft × 3 VA (includes ALL general receptacles)
```

### ❌ Violation 2: Using Actual Appliance Names for (B)(2)
```
Wrong: Toaster 1200W + Coffee Maker 900W + Microwave 1500W = 3,600 VA
Right: 2 circuits × 1,500 VA = 3,000 VA (fixed value, no nameplates)
```

### ❌ Violation 3: Applying Demand Factor to (B)(2) Before Summing
```
Wrong: Apply 40% to small-appliance circuits individually
Right: Sum ALL loads first, then apply demand factor to total
```

### ❌ Violation 4: Less Than 2 Circuits
```
Wrong: 1 small-appliance circuit for small kitchen
Right: MINIMUM 2 circuits per 210.11(C)(1) — no exceptions
```

## Bash Implementation (DEATHSTAR Oracle)

```bash
# 220.82(B)(2) Small-Appliance Circuits
echo "Number of small-appliance branch circuits (min 2 per 210.11(C)(1)):"
read -r SMALL
if (( SMALL < 2 )); then
    SMALL=2
    echo "Minimum 2 enforced by code. Adjusted."
fi

# 220.82(B)(3) Laundry Circuit
echo "Number of laundry branch circuits (min 1 per 210.11(C)(2)):"
read -r LAUNDRY
if (( LAUNDRY < 1 )); then
    LAUNDRY=1
    echo "Minimum 1 enforced by code. Adjusted."
fi

# Calculation
lighting=$(( SQFT * 3 ))
small_va=$(( SMALL * 1500 ))
laundry_va=$(( LAUNDRY * 1500 ))

raw_general=$(( lighting + small_va + laundry_va ))

# Add fixed appliances (100% - NO demand factor)
for app in "${APPLIANCES[@]}"; do
    va=$(echo "$app" | cut -d':' -f2)
    raw_general=$(( raw_general + va ))
done

# Demand factor
if (( raw_general > 10000 )); then
    demanded=$(( 10000 + (raw_general - 10000) * 40 / 100 ))
else
    demanded=$raw_general
fi
```
